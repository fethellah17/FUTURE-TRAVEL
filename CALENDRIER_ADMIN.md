# Calendrier Interactif - Dashboard Admin

## Nouvelles Fonctionnalités

### 1. Date Picker Interactif
- Remplace les champs de saisie textuelle par un calendrier interactif élégant
- Design cohérent avec le thème : fond blanc, accents en Bleu Roi
- Format d'affichage : DD/MM/YYYY (ex: 15/03/2026)
- Locale française pour les noms des mois et jours

### 2. Sélection de Plage de Dates
- Deux calendriers : Date de début et Date de fin
- Permet de sélectionner précisément la période du voyage
- Interface intuitive avec pop-up au clic

### 3. Calcul Automatique de la Durée
- La durée est calculée automatiquement si les deux dates sont sélectionnées
- Affichage en temps réel : "X jours"
- Le champ durée reste modifiable manuellement si nécessaire

### 4. Format de Date Automatique
- Génération automatique du format d'affichage : "15 Mar - 25 Mar 2026"
- Format français avec les abréviations des mois
- Prêt pour le stockage en base de données

### 5. Disponible dans les Deux Formulaires
- Formulaire d'ajout de voyage
- Formulaire de modification de voyage (modal)

## Utilisation

### Ajout d'un Voyage
1. Cliquer sur "Ajouter un Voyage"
2. Remplir les informations de base (Titre, Prix, Catégorie)
3. Cliquer sur les champs de date pour ouvrir le calendrier
4. Sélectionner la date de début et la date de fin
5. La durée et le format de date sont calculés automatiquement
6. Possibilité de modifier manuellement la durée si besoin
7. Enregistrer

### Modification d'un Voyage
1. Cliquer sur l'icône crayon (Pencil) à côté d'un voyage
2. Le modal s'ouvre avec les informations actuelles
3. Utiliser les calendriers pour modifier les dates
4. La durée se recalcule automatiquement
5. Enregistrer les modifications
